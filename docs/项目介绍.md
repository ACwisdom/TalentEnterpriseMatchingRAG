# 项目介绍

人才—企业智能匹配与报告生成系统（RAG）。以企业 Excel 库为数据源，构建 Chroma 向量索引，融合关键词、向量与 RRF 混合检索，从博士等人才简历中召回匹配企业；可接入小米 MiMo 生成匹配报告与人才画像，并导出 Word。支持 PDF、DOCX、TXT 简历解析，提供流水线脚本、FastAPI 网页上传、`/demo` 固定目录展示及本地静态演示页。可通过 Docker、Fly.io 部署，密钥由环境变量管理，适用于引才咨询、政企对接中的初筛与材料起草辅助。系统提示词与检索参数可通过配置文件与环境变量调整，便于二次开发与对接。高级人才画像可结合微信截图等多模态输入。详述见仓库 README。

（约 300 字。）
