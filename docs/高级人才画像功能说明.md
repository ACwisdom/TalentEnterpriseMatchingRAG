# 高级人才画像功能使用说明

## 功能概述

高级人才画像功能可以结合**简历文本**和**微信聊天截图**，生成深度人才分析报告，包括：
- 基本信息
- 研究方向
- 核心技能
- **差异化优势（深度分析）**
- **核心诉求（显性+隐性）**
- 潜在风险

## 配置要求

### 1. config.py 配置
```python
# MiMo API配置（已配置）

```

## 微信截图要求

### 格式要求
- **格式**：PNG（推荐）或 JPG
- **数量**：多张分段截图（按对话顺序）
- **内容**：微信聊天记录，包含与人才的对话

### 截图质量建议
- 确保文字清晰可见
- 包含日期时间信息
- 截取包含关键信息的对话段落
- 多张截图按时间顺序排列

### 示例截图路径
```
data/input/wechat/
├── screenshot1.png  # 第一段对话
├── screenshot2.png  # 第二段对话
└── screenshot3.png  # 第三段对话
```

## 输出格式

### Word报告结构
1. **标题**：人才-企业匹配报告
2. **生成时间**：自动添加
3. **高级人才画像**（如果启用）：
   - 一、博士人才画像
   - 1. 基本信息
   - 2. 研究方向
   - 3. 核心技能
   - 4. 差异化优势（深度分析）
   - 5. 核心诉求（显性+隐性）
   - 6. 潜在风险
4. **企业匹配结果**：
   - 高度匹配企业（≥70分）
   - 潜在匹配企业（40-70分）
   - 行业相关企业（<40分）
5. **匹配总结**：统计表格

### 高级人才画像特点
- **深度分析**：不只是罗列技能，而是结合背景、经验、成果进行解读
- **隐性诉求挖掘**：从聊天记录中推断未明说的需求
- **风险评估**：识别可能影响回国决策的因素
- **专业格式**：Markdown格式，自动转换为Word样式

## 常见问题

### Q1: 多模态API调用失败（400 Bad Request）
**A**: 确保截图文件存在且格式正确（PNG/JPG），文件大小不要太少（至少1KB）。

### Q2: 微信截图分析返回空结果
**A**: 确保截图中包含清晰的文字内容，避免模糊、反光、缺失关键区域的截图。

### Q3: 高级人才画像格式不正确
**A**: 检查 `word_exporter.py` 中的 `_add_advanced_profile()` 方法是否正常工作。

### Q4: 如何获取微信聊天截图？
**A**: 使用微信PC版，打开与人才的聊天窗口，使用截图工具（Ctrl+Alt+A）截取对话内容，保存为PNG格式。

### API请求格式（多模态）
```python
messages = [
    {"role": "system", "content": system_prompt},
    {"role": "user", "content": [
        {"type": "text", "text": "请分析以下微信聊天截图..."},
        {"type": "image_url", "image_url": {"url": "data:image/png;base64,..."}}
    ]}
]
```

### 多模态模型
- **模型名称**: `mimo-v2.5`
- **API端点**: `https://api.xiaomimimo.com/v1/chat/completions`
- **支持格式**: Base64编码的PNG/JPG图片
